# 🛰 AI Micro-Doppler Radar Detection System

## 📌 Overview
This project simulates a radar system that classifies signals as Drone or Bird using Machine Learning.

## 🧠 Features
- Real-time radar simulation
- Streamlit dashboard UI
- ML-based classification

## 📂 Project Structure
- app/ → frontend + backend
- model/ → training + model file
- data/ → datasets

## ▶ Run Project

cd app
streamlit run streamlit_app.py
